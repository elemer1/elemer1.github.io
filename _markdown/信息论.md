* $$G_{Max} = I(X;Y)$$。一个 Agent 的最大可靠财富增长率，等于 ta 从世界获取的信号与真实结局之间的互信息。
* Shannon 信道容量定理：$$C = \max_{p(y)} I(X; Y)$$。一个带噪信道的最大可靠传输速率，等于输入分布和输出分布之间的互信息最大值。
* 如果 agent 对某个标的的概率判断优于其他 agent 的平均判断，那么其长期收益率 G_Max 就是负数。它们的信道里没有 bit 流过，用 Kelly 的语言说就是 I(X;Y) = 0。很多人把亏损归因于”运气差”或“不努力”等原因，其实更准确的表述是”没有信息优势”。这是两个完全不同的诊断，后者才有修正的方向。你赚的钱，本质上来自信息。下注只是把 bit 兑成钱的那一步换算。没有 bit 流入，下再漂亮的注也是白忙。所以做任何有不确定性质的活动（交易、投资、谈判、择偶、创业中的重大资源分配），真正要持续投入的地方是上游——找到独家、可验证、会持续产出 bit 的信息源，而非下游的资源管理花样。Kelly Criterion 只是告诉你拿到信息后怎么最优地兑现它；它不会给你信息本身，信息得靠你自己去挖。
